1、shell简介

1.1、简介

shell命令解释过程：用户在命令行输入命令提交后，shell程序首先检测是否是内置命令，如果是，就通过shell内部的解释器将命令解释为系统调用，然后提交给内核执行；如果不是，shell会按照用户给出路径或者环境变量路径在硬盘寻找相应命令，然后调入内存，最后解释为系统调用，提交给内核执行。

1.2、语法分析

1.2.1、命令格式

command [options] [arguments]
  • command：命令名称
  • options：命令选项
  • arguments：命令参数

在shell一个命令行中，可以输入多个命令，用分号隔开

ls -al; cp 1.txt 2.txt

也可以在多行中输入一个命令，用”\“将一个命令持续到下一行

cp -i \
1.txt \
2.txt

1.2.2、通配符

1.2.2.1、* 匹配任意一个或多个字符

ls *.txt

上面命令列出当前目录下所有txt文件

ls -al /etc/*/*.conf

上面命令列出etc目录下所有子目录下以.conf结尾的文件，不包括etc目录下的conf文件

1.2.2.2、? 匹配任意单一字符

ls ab?.txt

列出当前目录下ab开头，随后一个字符任意字符的txt文件

1.2.2.3、[] 匹配任何包括在方括号内的单字符

ls /dev/sda[1-5]
或者 ls /dev/sda[12345]

1.2.3、shell重定向

重定向是指不使用系统默认的输入输出，而是重新指定，分为输入，输出和错误输出重定向

1.2.3.1、输入重定向

操作符有 < 和 <<

wc

用wc命令统计输入给他的文件的行数，单词数和字符数

<<告诉shell当前输入来自命令行中一对分隔号之间的内容

1.2.3.2、输出重定向

ps -ef >ps.txt

该命令将进程运行信息全部输入到了ps.txt文件，而不输出到屏幕，>默认会覆盖文件内容，而>>是追加

1.2.3.3、错误重定向

使用 2> 或者 2>>

1.2.4、shell管道

管道的作用就是将一个命令的输出当作下一个命令的输入，而不经过任何中间文件。

ls -al /etc/* | more

上面命令将etc目录以及其子目录下所有文件分屏显示

ps -ef | grep httpd|wc -l

该命令用来查看系统中正在运行的httpd进程，并统计httpd进程数

1.2.5、shell中的引用

1.2.5.1、转义字符\

mv abc\?\* abc

上面命令将abc?*重命名为abc

1.2.5.2、单引号’

将字符串放到一对单引号之间，那么字符串中所有特殊含义都会被忽略

mv 'C:\backup' backup

等价于

mv C\:\\backup backup

1.2.5.3、双引号”

和单引号作用基本相同，但仍有一些字符用双引号括起来也还是保留自己的特殊含义，如**$,\和`**

1.2.6、shell自动补全命令行

Tab键

2、系统管理与维护

2.1、ls命令

ls [选项] [路径或文件]

如

ls -l /home

列出home目录下文件及其子目录

2.2、pwd命令

显示当前工作目录

2.3、cd命令

改变当前工作目录

2.4、date命令

普通用户只能显示日期和时间，超级用户才能修改日期和时间

用指定格式显示日期和时间

date '+this date now is:%x, time is now:%X, thanks!'

修改系统时间

date -s 20210512
date -s 14:58
date -s "20210512 14:58"

2.5、passwd命令

设置用户密码

普通用户使用 passwd根据提示连续输入两次密码即可

超级用户可以使用 passwd [用户名]修改某个用户的密码

2.6、su命令

用于改变用户身份

su -

根据提示输入超级用户密码，即可切到超级用户（-是加载root环境变量，不加也可，可能有些命令无法执行）

2.7、clear命令

清除屏幕信息

2.8、man命令

显示指定命令的帮助信息

man ls

上面命令可以得到ls命令的帮助信息

2.9、who命令

显示目前登陆到系统的用户

who -r    #查询系统处于什么运行级别
who -buT  #显示系统最近启动日期，以及当前每个用户的登陆详情、终端状态

2.10、w命令

显示登陆到系统的用户信息

w

2.11、uname命令

显示操作系统相关信息

uname -a

2.12、uptime命令

输出系统任务队列信息

2.13、last命令

列出目前和过去登陆系统的用户相关信息

2.14、dmesg命令

显示开机信息

2.15、free命令

显示系统内存状态，包括物理内存，虚拟内存，共享内存和系统缓存

free -m

2.16、ps命令

显示系统进程在瞬间的运行状态

显示系统所有进程

2.17、top命令

实时显示系统中各个进程的资源占用情况

3、文件管理与编辑

3.1、mkdir命令

创建一个目录

mkdir -p /home/user1/user2

在home目录下创建嵌套目录user1/user2，如果user1不存在会同时创建user1和user2目录

3.2、more命令

more命令读取文本文件时，每次显示一屏，并且在每屏后暂停，同时在屏幕底部显示单词more，按enter键会显示下一行

3.3、cat命令

将文件的内容输出到标准输出

cat -Ab Kconfig

查看Kconfig文件的内容，同时将制表符显示为”^I”， 对非空行进行编号输出

cat file1.txt file2.txt >file3.txt

以上命令将file1.txt和file2.txt内容合并后输出到file3.txt

3.4、diff命令

用来比较文件的差异

diff [选项] file1 file2

3.5、grep命令

Linux下的文本过滤工具，grep根据指定的字符串，对文件的每一行进行搜索，如果找到了该字符串，就输出该行

grep rpm install.log

上面命令列出install.log文件中含有RPM字符串的行

3.6、rm命令

用来删除某个目录及其下的所有文件和子目录。对于链接文件，只是断开了链接，原文件保持不变。

rm [选项] 文件或者目录

3.7、touch命令

改变指定文件的访问时间和修改时间，若文件不存在，则创建该文件

touch test.txt  #创建test.txt文件
touch -a test.txt  #改变文件访问时间为系统当前时间

3.8、ln命令

用来在文件或者目录之间创建链接。链接分硬链接和符号链接（软链接）两种。ln默认创建硬链接

硬链接：多个文件指向同一个iNode文件节点

软链接：类似Windows快捷方式，是一个指向真正文件位置的符号链接

ln [选项] 源文件目标链接名
ln /etc/inittab /opt/backup  #将/etc/inittab文件硬链接到/opt/backup目录下
ln -s /etc/inittab /opt  #将/etc/inittab文件符号链接到/opt下

3.9、file命令

显示文件的类型

file /ect/init.d/halt  #显示为shell可执行文件
file /bin/sh    #显示为符号链接文件
file file1.txt #显示为ASCII文本文件
file /dev/had #显示为块设备文件

3.10、cp命令

用来将给的文件或目录复制到另一个文件或者目录中。

cp [选项] 源文件或目录 目标文件或目录
cp -r ./* /tmp  #将当前目录下所有文件复制到/tmp目录下

3.11、find命令

在指定路径下查找指定文件

find path-name[-options] [-print -exec -ok命令{} \;]

3.12、split命令

用来分割文档，将一个文件分成数个

split [选项] [input_file] [output_file]
split -b 10M access_log access_log_bak  #将access_log文件进行分割，设定每个分割文件大小为10M

3.13、mv命令

将文件或目录改名或者将文件由一个目录移入另一个目录

mv [选项] 源文件或目录 目标文件或目录
mkdir myDocument
touch Document
ls -la
mv Document myDocument
mv myDocument myDocument_1
ls -la  #此时只有my_Document_1目录

4、压缩与解压

4.1、zip/unzip命令

zip [选项] 压缩文件名 需要压缩的文档列表
unzip [选项] 压缩文件名

zip -9r /opt/etc.zip /etc  #将etc目录下所有文件和目录进行压缩，并设置压缩级别为最高
unzip -o /opt/etc.zip -x etc/inittab -d /etc  #将/opt/etc.zip压缩文件全部解压到/etc目录下，除了etc/inittab文件，并在解压过程出现相同文件就覆盖

4.2、gzip/gunzip命令

将一般的文件进行压缩或者解压，压缩文件默认是.gz文件

gzip [选项] 压缩（解压缩）的文档名

zip -r /opt/etc.zip /etc   
gzip -9v /opt/etc.zip   #对etc.zip文件进行gzip压缩，并设置压缩级别为9

4.3、bzip2/bunzip2命令

对文件进行压缩和解压缩，生成bz2压缩包，类似于gzip/gunzip

4.4、tar命令

对文件或目录进行打包归档

tar [主选项+辅助选项] 文件或者目录
tar -cvf /opt/etc.tar /etc  #将etc目录下所有文件打包，并显示打包详细信息
tar -zcvf /opt/etc.tar.gz /etc #将etc目录打包成etc.tar文件，并在打包完成后进行gzip压缩，最终打包成tar.gz文件

4.5、dd命令

转换或者复制文件，也可以对设备进行备份

dd if="input_file" of="output_file" bs="block_size" count="number"

dd if="/etc/inittab" of="/opt/inittab.bak"  #将/etc/inittab文件备份到/opt/inittab.bak文件中
dd if=/dev/sda2 of=/opt/sda2.back  #将/dev/sda2整个分区进行备份，保存备份文件到/opt目录

4.6、cpio命令

通过重定向的方式对文件进行打包、备份、还原、恢复的工具，可以解压以.cpio或者.tar结尾的文件

cpio [选项] >文件名或者设备名
cpio [选项] <文件名或者设备名
find /etc -type f | cpio -ocvB >/opt/etc.cpio   #将/etc下的所有普通文件都备份到/opt/etc.cpio中

5、磁盘管理与维护

5.1、df命令

检查Linux系统磁盘空间占用情况

df [选项]
df -hT  #以人性化方式显示当前磁盘分区情况，并显示文件系统类型

5.2、du命令

显示文件或目录占用的磁盘空间情况

du [选项] 文化或目录
du -sh /etc  #显示 /etc目录的总大小

5.3、fsck命令

检查文件系统并尝试修复错误

fsck [选项] [-t <文件系统类型>] [设备名]
fsck -rV -t ext3 /dev/sda10 #检查并尝试修复出问题的磁盘分区/dev/sda10, 修复时进行询问让用户确认并决定处理方式，显示修复详细过程

5.4、sync命令

强制把内存中的数据写回硬盘，以免数据丢失

5.5、eject命令

用来退出抽取式设备，如光驱，磁带

5.6、mount/umount命令

挂载/卸载指定的文件系统

6、网络设置与维护

6.1、ifconfig命令

配置网络或者显示当前网络接口状态，必须以root用户执行

ifconfig [选项] [interface] [inet|up|down|netmask|addr|broadcast]

6.2、scp命令

secure copy的缩写，用户将文件或目录从一个Linux系统复制到另一个Linux系统

6.3、netstat命令

显示本机网络连接、运行端口和路由表等信息

netstat [选项]
netstat -rn   #显示当前系统的路由信息
netstat -an #显示系统当前所有有效的TCP连接

6.4、traceroute命令

显示网络数据包传输到制定主机的路径信息，追踪数据传输路由状况。

traceroute [选项] [远程主机名或者IP地址] [数据包大小]
traceroute -s 192.168.100.137 -w 10 www.baidu.com 100   #在指定超时时间10S内检测不到某个路由节点的回应信息，就输出*

6.5、telnet命令

通过telnet协议与远程主机通信或者获取远程主机对应端口的信息

telnet 主机名或者IP地址端口
telnet 192.168.60.123 23  #连接192.168.60.123主机的23端口

6.6、wget命令

用于从网络上下载某个软件

wget [要下载软件的网址]
wget https://mirrors.edge.kernel.org/pub/linux/kernel/v5.x/linux-5.0.3.tar.xz  #下载linux-5.0.3内核

7、vi

有命令模式和输入模式两种模式

vi filename  #打开或者新建一个文件，并将光标置于第一行的首部
ESC   #返回命令行模式
i #在光标前输入文字
a #在光标后输入文字
/abc  #在文件中向前查找字符串abc
?abc  #在文件中向后查找字符串abc
n  #在同一方向重复上次的搜索命令
N  #在反方向重复上次搜索命令
:wq  #保存并退出vi
:q  #不保存退出vi
